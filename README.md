# QA-Portfolio: Demo Webshop Testing
Dieses Projekt dient zur praktischen Anwendung der **ISTQB Foundation Level** Prinzipien. Ich nutze diesen Webshop, um professionelle Testdokumentation zu erstellen und den Software-Testlebenszyklus (STLC) zu durchlaufen.

## 📋 Projektübersicht
- **Testobjekt:** [Demoblaze / Demo Webshop]
- **Ziel:** Qualitätssicherung der Kernfunktionen (Suche, Warenkorb, Checkout).
- **Methodik:** Manuelles Testen nach ISTQB-Standard.

## 📂 Dokumentation
- [**Testplan**](Testplan.md): Strategie, Umfang und Risiken.
- [**Testfälle**](Testcases.md): Detaillierte Schritt-für-Schritt-Anleitungen inkl. Testdaten.
- [**Fehlerberichte (Bug Reports)**](Bugreports.md): Dokumentation gefundener Abweichungen.

## 🛠 Angewandte Techniken
- **Black-Box-Testen:** Äquivalenzklassenbildung und Grenzwertanalyse.
- **Fehler-Management:** Priorisierung und Klassifizierung von Defects.
- **Traceability:** Verbindung zwischen Testfällen und Anforderungen.
